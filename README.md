# BRISAMARINA Landing Page

Este repositorio contiene el código fuente para la landing page del proyecto BRISAMARINA.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

* `css/`: Contiene los archivos de hojas de estilo (ej. `style.css`).
* `js/`: Contiene los archivos de interactividad y scripts de JavaScript (ej. `script.js`).
* `img/`: Carpeta destinada a almacenar todos los recursos gráficos e imágenes.
* `main.html`: Archivo principal de la página web.
